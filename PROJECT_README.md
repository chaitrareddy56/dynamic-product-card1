# Dynamic Product Card

Dynamic Product Card is a reusable React component that displays product image, title, description, and pricing with sale indication. It includes interactive Add to Cart and Favorite toggle features and is fully responsive.

## Features
- Product image, title, and truncated description
- Sale price with strikethrough original price
- Add to Cart button with 2-second toggle
- Favorite (heart) toggle
- Responsive design for mobile and desktop

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173
